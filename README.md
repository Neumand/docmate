# Docmate

Docmate is an AI-powered document assistant.

It's a SaaS portfolio project that I built to learn some new technologies and tools:

- Next.js 13+ (App Router, etc.)
- Prisma
- tRPC
- shadcn/ui
- AI tools (langchain, Pinecone, OpenAI API, etc.)
